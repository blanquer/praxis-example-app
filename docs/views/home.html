Blogger Example API

Blogger is a simple API that allows simple management of Blogs, Posts and their Authors.

While originally inspired by the Google Blogger API, it is only meant to be a simple (and nowhere near feature-complete) example of a Praxis application, simply to demonstrate some of the various available features of the framework.

Authentication

There is currently no authentication mechanism for using this API.

Using the API

Currently, the Blogger API only offers a 1.0 version, exposed as a path prefix.

You can issue requests directly against the Blogger API with your favorite HTTP tool. This is an example of show request for the "default" view of a Blog (assuming you have the app running on port 9292 of localhost):

Request using Curl: 
curl -X GET https://localhost:9292:/api/v1.0/blogs/1
